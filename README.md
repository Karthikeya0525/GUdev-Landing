# UI&UX design: AI-Powered PRD Generator

UI&UX design is a SaaS web application that turns startup ideas into professional, investor-ready Product Requirements Documents (PRDs) and Landing Page previews in seconds.

## 🚀 Features

- **Minimalist White UI:** Clean, distraction-free interface inspired by Apple design principles.
- **AI-Powered Analysis:** Validates ideas, structures them professionally, and generates a comprehensive PRD.
- **Founder Workspace:** A professional environment with a JSON editor and live PRD preview.
- **Web Preview:** Instantly generate a modern landing page for your startup idea.
- **PDF Generation:** Automatically generates a beautifully formatted, downloadable PDF.

## 🛠 Tech Stack

- **Frontend:** Next.js (App Router), TailwindCSS, Framer Motion, Monaco Editor.
- **Backend:** Next.js API Routes (Serverless), Groq AI (Llama 3.3).
- **Database:** Supabase (PostgreSQL).
